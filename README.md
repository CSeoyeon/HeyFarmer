# HeyFarmer

계절, 농산물 시세비교 웹 사이트 

## 담당 역할  
- UI 
- 회원가입/로그인/로그아웃 구현 
- 게시판 글 작성, 수정, 삭제, 검색 구현
- 기타 세부 기능 

## 구현 한계점 
- 농산물 검색 결과 확인 불가
- 원인: csv 파일과 웹 연동 코드 실패, 잘못된 쿼리문 작성
